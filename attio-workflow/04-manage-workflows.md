# Manage Workflows (Attio Reference)

Source: https://attio.com/help/reference/automations/workflows/manage-workflows

## Manage Access to Workflows

- Only workspace admins and members with full workflow access can modify permissions
- Four access levels: **Full access**, **Read and write**, **Read only**, **No access**
- Workspace access = default; team access overrides it; member access takes priority over both

## View Workflows (List Page)

### Sort Options
- Creation date
- Last published
- Number of runs
- Name

### Filter Options
- Filter by creator

### Group By Options
- Creation date
- Status
- Last published
- Creator
- Trigger

### Other Features
- Add workflows to favorites for quick access
- Toggle archived workflows visibility

## Archive, Restore, and Delete Workflows

- **Archive** pauses workflows; accessible via three-dot menu
- **Restore** previously archived workflows (remains paused until published)
- **Delete** removes workflows; "ongoing runs are immediately cancelled"
- Only workflows without prior runs can be deleted

## Admin Settings

- Admins can access workflow settings from three locations
- Manage permissions, pause, archive, and duplicate workflows

## Manage Workflow Credits & Notifications

- Set credit quota per run (default: 50 credits)
- Enable/disable email notifications for failed runs
- Receive maximum one email per day per unique failure reason
